Getting started

Sign up for opensauce:

  • http://saucelabs.com/opensauce

Install zuul:

npm i -g zuul

Configure zuul:

# ~/.zuulrc
sauce_username: me
sauce_key: abc12348-e3e2-...

Add .zuul.yml to your project:

# .zuul.yml
ui: mocha-bdd
browsers:
  - name: chrome
    version: latest
  - name: ie
    version: 6..latest
  - name: firefox
    version: latest

Try to run tests:

zuul test/test.js
zuul --local test/test.js
